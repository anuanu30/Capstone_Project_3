## Overview

This project builds a machine learning model to predict whether a customer will churn within the next 60 days.

The workflow includes:
- Data preprocessing
- Leakage prevention
- Baseline and advanced models
- Threshold optimization
- Model evaluation
- SHAP explainability
- Error analysis
- Model card documentation
